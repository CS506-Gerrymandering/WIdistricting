The Project

Originally proposed by Becky Andresen, this project stems from the need for a free and open-source library for the tools necessary for gerrymandering detection. The application (this site) is meant to educate people about some of the metrics used to detect gerrymandering as well as to provide an example to developers of how to use the library—every function in the library is used in this application (see developer guide for more).

The Team

The team consists of six CS students at the University of Wisconsin—Madison. Everyone on the team selected this project from nearly 30 proposals because they believed this was a way to use their skills to support free and fair elections in the US.
